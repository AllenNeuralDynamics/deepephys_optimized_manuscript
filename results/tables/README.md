# Collated tables (manuscript inputs)

Generated by [`../../code/figures/collate.py`](../../code/figures/README.md) from the manifest and
`../scores/`. Each is written as both `.csv` (data) and `.md` (a Markdown table the manuscript
`{include}`s). **Do not edit by hand** — re-run `collate.py`.

| file | contents | used by |
|---|---|---|
| `master_table.{csv,md}` | one row per run: config, loss, mean-over-units of every metric, ±SD where replicated | Results (T1) |
| `perunit_amp.{csv,md}` | `amp_ratio`, units (rows, by baseline d′) × models (cols) | Appendix B (T4) |
| `perunit_dprime.{csv,md}` | `dprime_deep`, units × models | Appendix C (T5) |
| `perunit_dprime_delta.{csv,md}` | `dprime_deep − dprime_raw`, units × models | Appendix C heatmap |
| `noise_floor.{csv,md}` | σ of d′ and amp over seed replicates, per config | Methods / T2 |

Until scoring lands these are absent; the corresponding manuscript `{include}`/figure directives are
commented out and get uncommented as the tables appear.
